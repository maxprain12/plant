# Sprites de Manny (fantasmito)

Especificaciones para los sprites del mascota fantasmito Manny.

## Formato

- **Tamaño**: 32x32 px o 64x64 px
- **Formato**: PNG con canal alpha (transparencia)
- **Estilo**: Silueta blanquecina/sem-transparente, ojos expresivos, forma de fantasma clásico

## Archivos requeridos

| Archivo   | Uso                                      |
|-----------|------------------------------------------|
| idle.png  | Estado por defecto (fantasmito en reposo)|
| walk-1.png| Frame 1 de animación al moverse           |
| walk-2.png| Frame 2 de animación al moverse           |
| wave.png  | Hover antes de abrir chat (saludo)       |
| think.png | Cuando Many está respondiendo             |

## Sugerencias de diseño

- Forma de "sábana" o silueta fantasma suave
- Ojos amigables (no aterradores)
- Expresión carismática y divertida que refleje la personalidad de Manny
- Colores: blanco, gris claro o tonos azulados translúcidos
- Evitar detalles complejos; mantener legible a 32x32 px
